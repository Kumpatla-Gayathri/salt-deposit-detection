# 🧂 Salt Deposit Identification Using Deep Learning

## 📌 Overview
A deep learning model to detect and segment salt deposits 
from seismic images using U-Net + ResNet34 architecture.

## 🏆 Results
- **IoU Score:** 0.7852
- **Training Images:** 3400
- **Validation Images:** 600
- **Epochs:** 30

## 🌐 Live Demo
👉 [Try it here](https://huggingface.co/spaces/GayathriKumpatla/salt-deposit-detector)

## 🛠️ Tech Stack
- Python
- PyTorch
- U-Net + ResNet34
- Albumentations
- Gradio
- Google Colab (T4 GPU)

## 📊 Dataset
[TGS Salt Identification Challenge — Kaggle](https://www.kaggle.com/c/tgs-salt-identification-challenge)

## 🔄 Workflow
1. Data Preprocessing & Augmentation
2. U-Net + ResNet34 Model Building
3. Training with Dice + BCE Loss
4. Evaluation with IoU Score
5. Deployment on Hugging Face Spaces
